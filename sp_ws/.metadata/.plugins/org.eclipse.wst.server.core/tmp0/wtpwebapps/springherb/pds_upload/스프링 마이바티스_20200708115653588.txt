마이바티스 
**데이터베이스 연결 정보와 
**매퍼(SQL을 선언한 별도의 XML이나 애노테이션을 가진 인터페이스)의 위치를 지정함

설정 파일이 준비되면 설정 파일의 내용을 갖는 객체를 생성해야 함
이 객체를 생성하기 위해서는 설정 파일을 파싱하고 로드하는 과정

***SqlSessionFactory 애플리케이션 내에서 한 개만 생성되어야 함
SqlSession : 하나의 커넥션
매퍼XML - 매핑구문

동적 => 해쉬맵 => 원시적 파라미터 쓸 수 없음 map이나 vo

스프링 - 마이바티스 설정 6가지
-데이터소스 : db연결정보
-트랙잰션관리자
-어노테이션을사용한트랜잭션처리
-어노테이션이붙은애들빈동록(자동스캔) 
	@Service (@Transactional처리함) @Repository (DAO임) @Controller @Component
-마이바티스설정 sql세션팩토리빈 별칭/마이배티스설정확인/매퍼위치
-마이바티스설정 sql세션템플릿
